# Tech Stack

AgentClinic will use server-side TypeScript with Node.js.

Recommended framework:
- Express for a minimal, popular, TypeScript-friendly backend.

Database:
- SQLite for simple, file-based storage during development and demos.

Additional stack considerations:
- TypeScript for safety and developer productivity.
- Express middleware for routing, JSON handling, and web APIs.
- Modern browser compatibility for an attractive frontend experience.
